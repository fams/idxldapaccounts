Create group screen

This screen is a simple form where you may enter the essential parameters for a group creation

Parameters

  • Organization:

    This is the Organization Unit under which the group will be created, it may be selected within all OU that reside under the "LDAP groups search base" configured OU. By default, the selected OU is the one that was active in the groups listing screen when you clicked on the "Add group button".

  • Name:(cn)

    A name for the new group (that may contain space characters)

    This parameter must be unique in the LDAP.

  • Gid number:(gidNumber)

    A unique (in the LDAP) number greater than the configured "minimum gid number" parameter, a valid one will be proposed to you when you first access the form and if you change it to a non valid one, another valid one will be proposed to you again.

  • Description:(description)

    A brief description for this group.

Group creation

When you think the group's parameters are corrects, click on the "Create group" button at the bottom of the screen to effectively create the group.

If some parameters are incorrect, an error screen is displayed and you may correct you errors by redisplaying the creation form with the "Back" button of you browser.

If there was no error, an entry is created in the LDAP for this group.

If group creation was successfull, you access the edition screen for this group which displays a message confirming the success of the operation.
