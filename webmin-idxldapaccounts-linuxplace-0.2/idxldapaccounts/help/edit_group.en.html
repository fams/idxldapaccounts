Edit group screen

This screen allows you to view/modify a group's parameters.

A menu gives you access to two parameters categories: "General" and "Members".

Common features

Two elements are common to the twoo screens namely:

  • delete group button:

    When clicking on it, you will go to a confirmation page for completly removing the corresponding user from LDAP.

  • Apply changes button:

    This button located at the bottom of the screen allow you to commit the changes you made on the screen, if you made changes on a screen and live it without applying them, your modification are lost.

    Once you applied your changes and if no error occured, the same screen is redisplyed with the new updated parameters (no success message appears).

    If an error occured during the modification process, an error screen is displayed and you may redisplay the screen by clicking on the "Back" button of you browser and then correct errors (if these where only parameters errors).

    General Parameters

    • Organization:

      This is the Organization Unit under which the group was created, it is not editable.

    • Gid number:(gidNumber)

      A unique (in the LDAP) number greater than the configured "minimum gid number" parameter, this parameter is not editable.

    • Description:(description)

      A brief description for this group.

      Members

      This screen displays a listing of the various users belonging to the group, users which have this group as primary group are shown with a green text ("primary") and are not removable from this menu: you have to change the user primary group from the "General" screen of the user edition menu and then comme back to remove it from the group.

      Other users are removable (that is users won't belong to the anymore) by clicking on their respective "remove" button in the "Remove member from this group" column.

      When clicking on a user name in the list, you will jump to the user edition screen.

      You may add one or many new users to the group by clicking on the "Select members" button, a users listing window will then appear with a search form to search specific users, each user has a "Add button" that you use to append the user to the text area, when you finished your selection, close the users window and click on the "Add selected members to this groups" button to add all users listed in the text area to the groups.
