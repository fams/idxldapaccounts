Users screen

The main users screen presents you with a search form, add and delete buttons and users listing.

Display options

By default, users are ordered by uid, (called "name" through the interface), but you can change this by clicking on column headers that allow it (like the "Number" header).

When no search parameter is given (that is when first arriving to this screen or when clicking on the "Show complete list button"), the maximum number of users is displayed, this number is specified in the module configuration as the "maximum number of results displayed by a LDAP search" parameter.

If the number of users to be displayd exceeds the "maximum number of list items displayed per page" configuration parameter, results are displayed on multiple pages with a menu allowing direct access to each page.

Finally, if the "hide disabled users" configuration parameter is set, only users with enabled account will appear in the list. If this parameter is set to "no", a "Status" column will say if a user account is disabled.

Search

The search form allow to filter users based on the attribute specified in the "search by" select box.

On non numeric attributes, you can use LDAP wildcards in the search text box.

Search is always performed under the LDAP Organizational Unit specified by the "LDAP users search base" configuration parameter, but you can restrict it to a sub OU by selecting it in the OU tree displayed at the left of the search form.

When a search result is displayed, you may return to the full users listing by clicking on the "Show complete list" button that will then appear under the search form.

Adding users

Under the search form, you will find a "Add user" button which leads you to the user creation form. A specific help screen is accessible from this screen.

Editing users

The first column of the users listing displays users uid as links, when clicking on such a link, you will go to the edition screen for the corresponding user where you may modify his parameters. A specific help screen is accessible from this screen.

Deleting users

The last column of the users listing provides a "delete" button for each user, when clicking on it, you will go to a confirmation page for completly removing the corresponding user from LDAP.
